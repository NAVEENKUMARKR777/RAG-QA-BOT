# User Guide

## Uploading Documents
1. Open the application interface (`http://localhost:8501`).
2. Upload a PDF or text document using the provided file uploader.
3. The system will process the document and extract embeddings for each section.

## Asking Questions
1. After uploading a document, you can enter a query in the text box.
2. Click the "Ask" button.
3. The system will retrieve relevant information from the uploaded document and generate a coherent answer.

## Viewing Results
- The interface displays both the relevant document sections and the generated answer.
- You can upload multiple documents and continue querying the bot.
